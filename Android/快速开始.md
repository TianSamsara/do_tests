## 📱 打包为Android APK

### 方法一：使用Buildozer（推荐）

**前提条件**：需要Linux/Mac环境或使用WSL（Windows Subsystem for Linux）

1. **安装Buildozer**：
```bash
pip install buildozer
```

2. **在Android目录下执行**：
```bash
cd F:\kaifa\pythons\rj\Android
buildozer android debug
```

3. **生成的APK位置**：
```
bin/*.apk
```

### 方法二：使用Google Colab在线打包

如果本地配置困难，可以使用Google Colab：

1. 将整个项目上传到GitHub
2. 在Google Colab中运行buildozer
3. 下载生成的APK文件

### 方法三：使用第三方打包服务

- **AppVeyor**
- **GitHub Actions** (需要配置Android SDK)

## ⚙️ 配置说明

### buildozer.spec 关键配置

```ini
[app]
title = 刷题系统                  # 应用名称
package.name = quiz_system       # 包名
package.domain = com.quiz        # 域名
source.dir = quiz_app            # 源代码目录
version = 1.0.0                  # 版本号
requirements = python3,kivy      # 依赖
orientation = portrait           # 屏幕方向
android.api = 33                 # Android API级别
android.minapi = 21              # 最低API级别
```

## 🐛 常见问题

### Q1: 运行时提示找不到Kivy？
**A**: 安装Kivy：
```bash
pip install kivy==2.3.0
```

### Q2: 题库加载失败？
**A**: 检查：
1. JSON文件格式是否正确
2. 文件编码是否为UTF-8
3. 文件是否在question_banks目录下

### Q3: 打包APK失败？
**A**: 
1. 确保安装了Java JDK 8-11
2. 确保Android SDK配置正确
3. 查看buildozer日志排查具体错误

### Q4: 中文显示乱码？
**A**: 
1. 确保JSON文件使用UTF-8编码
2. Kivy会自动处理中文显示
